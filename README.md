# ⚡ Curso de PHP — Exercícios e Aprendizado

Bem-vindo(a) ao meu repositório de estudos de PHP! Este espaço foi criado para armazenar todos os códigos, exercícios práticos e projetos desenvolvidos por mim durante a minha jornada de aprendizado na linguagem PHP, integrando conceitos modernos de back-end com interfaces web limpas e estilizadas.

---

## 🚀 O que estou estudando?

Até o momento, o repositório cobre os seguintes fundamentos da linguagem:

* **Variáveis e Tipagem Dinâmica:** Manipulação de dados e uso correto de escopos.
* **Estruturas de Controle:** Tomadas de decisão utilizando condicionais complexas com `if`, `else if`, `else` e `switch/case` (com o uso correto de travamentos com `break`).
* **Estruturas de Repetição:** Controle de loops e laços com `for`, `while`, `do while` e `foreach`.
* **Estruturas de Dados:** Manipulação de Arrays simples (indexados) e Arrays Associativos (chave/valor).
* **Programação Orientada a Objetos (POO):** Criação de classes, propriedades (`public`), métodos, instanciação de múltiplos objetos independentes e manipulação do ciclo de vida dos objetos com métodos mágicos (`__construct` e `__destruct`).

---

## 💻 Destaques do Projeto

### Painel de Controle de Estruturas (`PHP Ex003`)
Um dos principais exercícios desenvolvidos conta com uma interface web moderna construída do zero, utilizando:
* **HTML5 semântico** integrado com tags de renderização do PHP.
* **CSS Moderno:** Layout responsivo, tipografia elegante (*Inter* via Google Fonts), gradientes vibrantes no fundo e nos títulos, e efeitos de desfoque de vidro (*Glassmorphism* com `backdrop-filter`).
* **Visualização em tempo real:** Blocos de códigos estilizados para simular editores de código e caixas de output para exibir o resultado lógico direto na tela.
